# Layout路径点编辑器

一个可视化工具，用于编辑Auto-Maple项目中的路径点文件。

## 功能特点

- **可视化编辑**：通过地图图片直观地添加和查看路径点
- **文件操作**：支持打开、保存和另存为Layout文件
- **坐标显示**：实时显示鼠标在地图上的相对坐标
- **简单易用**：界面友好，操作直观

## 安装依赖

确保你已经安装了以下依赖：

- Python 3.x
- OpenCV (`pip install opencv-python`)
- Pillow (`pip install Pillow`)
- Tkinter (Python标准库)

## 使用方法

1. **启动工具**：在项目根目录运行
   ```bash
   python layout_editor_GUI/layout_editor.py
   ```

2. **加载地图**：点击"加载地图"按钮，选择对应的地图图片（建议使用项目中 `assets/minimaps/` 目录下的地图图片）

3. **打开现有Layout**：点击"文件"菜单中的"打开"，选择Layout文件（如 `resources/layouts/kanna/dcup2`）

4. **添加路径点**：
   - 点击"添加点"按钮进入添加模式
   - 在地图上点击鼠标左键添加路径点
   - 路径点会以绿色圆点显示

5. **保存修改**：点击"保存"按钮保存修改后的Layout文件

6. **清除所有点**：点击"清除所有点"按钮可以清空当前布局

## 注意事项

- 删除点功能暂未实现，因为四叉树结构的删除比较复杂
- 保存时会直接覆盖原文件，建议先备份原文件
- 坐标系统使用相对坐标（0-1范围），与游戏内坐标系统一致

## 技术说明

- 使用Tkinter创建GUI界面
- 使用OpenCV处理图像
- 直接使用项目中的Layout类来处理路径点数据
- 保持与原项目的文件格式兼容
